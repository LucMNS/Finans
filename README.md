# 💰 Finans — Controle Financeiro Pessoal

O **Finans** é uma aplicação web moderna de gestão financeira, focada em performance, design responsivo e experiência do usuário (UX). Este projeto demonstra competências em desenvolvimento Full-Stack, integração com serviços de nuvem e arquitetura de software segura.

---

## 🔗 Demonstração Online

O projeto está publicado e pode ser acessado neste link:
👉 **https://finans-controle.vercel.app/**

---

## 🚀 Tecnologias Utilizadas

A stack tecnológica foi selecionada para garantir escalabilidade e agilidade:

- **React + Vite**: SPA rápida com suporte a Fast Refresh.
- **Supabase**: Autenticação e Banco de Dados PostgreSQL (BaaS).
- **Tailwind CSS**: Estilização responsiva baseada em utilitários.
- **Framer Motion**: Animações fluidas e reatividade visual.
- **Recharts**: Visualização de dados financeiros em gráficos interativos.
- **Lucide React**: Biblioteca de ícones consistente.

### Pré-requisitos
- Node.js (v18 ou superior)
- Uma conta no [Supabase](https://supabase.com/).

---

## ✨ Funcionalidades Principais

- **Autenticação Completa**: Login, cadastro e recuperação de conta via Supabase.
- **Dashboard em Tempo Real**: Resumo de saldo, despesas totais e saúde financeira.
- **Interface Otimista (Optimistic UI)**: Transações refletidas instantaneamente na UI antes da confirmação do banco.
- **Análise Temporal**: Filtros dinâmicos por mês e ano para histórico financeiro.
- **PWA (Progressive Web App)**: Instalável em dispositivos móveis e desktop.
- **Modo Escuro**: Suporte a temas claro/escuro com persistência de estado.
- **Interface Responsiva**: Design adaptado para smartphones, tablets e computadores.

---

## 🔒 Segurança

A segurança é tratada como prioridade no desenvolvimento:
- **Environment Variables**: As chaves sensíveis são injetadas em tempo de execução e nunca expostas no código público.
- **Git Hygiene**: O arquivo `.env` está configurado no `.gitignore` para prevenir vazamentos de credenciais.
